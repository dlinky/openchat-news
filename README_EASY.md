# ChatDigest — 쉽게 이해하는 사용 가이드

> 카카오 오픈채팅 대화를 AI가 정리해서 뉴스레터처럼 보여주는 서비스입니다.

---

## 이 서비스가 하는 일

오픈채팅방에서 매일 수백 개의 메시지가 오가지만, 다 읽기엔 너무 많죠.
ChatDigest는 채팅 내역 파일을 업로드하면 AI(Google Gemini)가 주요 내용을 자동으로 요약해서
**일간 / 주간 / 월간** 뉴스레터 형식으로 보여줍니다.

---

## 화면 구성

```
┌─────────────────────────────────────────────────────┐
│  왼쪽 사이드바          │  오른쪽 본문 영역           │
│                         │                             │
│  [파일 업로드]          │  선택한 날짜/주/월의        │
│  [요약 생성]            │  뉴스레터가 표시됩니다.     │
│                         │                             │
│  ▼ 일간                 │                             │
│    2026-03-05           │                             │
│    2026-03-04           │                             │
│  ▼ 주간                 │                             │
│    2026년 9주차         │                             │
│  ▼ 월간                 │                             │
│    2026년 2월           │                             │
└─────────────────────────────────────────────────────┘
```

### 왼쪽 사이드바

- **파일 업로드** 버튼: 카카오톡 대화 내역 파일(.txt)을 서버에 저장합니다.
- **요약 생성** 버튼: 저장된 대화를 AI가 분석해서 요약본을 만듭니다.
- **일간 / 주간 / 월간 목록**: 요약이 완료된 날짜/주/월 링크 모음. 클릭하면 본문에 표시됩니다.

### 오른쪽 본문

선택한 기간의 뉴스레터가 표시됩니다.
AI가 대화 내용을 토픽별로 묶어서 제목과 요점을 정리해줍니다.

---

## 사용 순서

### 1단계: 카카오톡에서 대화 내보내기

1. 카카오톡 오픈채팅방 입장
2. 우측 상단 메뉴(≡) 탭
3. **대화 내보내기** 선택
4. 텍스트 파일(.txt)로 저장

### 2단계: 로그인

웹 주소로 접속하면 비밀번호 입력 화면이 나옵니다.
설정된 비밀번호를 입력하고 로그인합니다.

### 3단계: 파일 업로드

1. 사이드바 상단 **파일 업로드** 버튼 클릭
2. 팝업 창에서 .txt 파일을 드래그&드롭하거나 직접 선택
3. **미리보기** 확인:
   - 새로 추가될 날짜 목록 (초록색)
   - 이미 저장된 날짜라 건너뛰는 목록 (회색)
4. **업로드 확인** 버튼 클릭

> 같은 날짜의 데이터가 이미 있으면 덮어쓰지 않고 그냥 넘어갑니다.
> 파일의 마지막 날짜(아직 끝나지 않은 날)는 자동으로 제외됩니다.

### 4단계: 요약 생성

1. 사이드바 **요약 생성** 버튼 클릭
2. **일간 / 주간 / 월간** 중 원하는 유형 선택
3. 날짜/주/월 선택
4. **생성** 버튼 클릭
5. AI 분석이 진행되는 동안 진행 상황이 실시간으로 표시됩니다.

> 일간 요약을 먼저 생성해야 주간, 월간 요약도 만들 수 있습니다.

### 5단계: 읽기

왼쪽 사이드바의 목록에서 날짜를 클릭하면 해당 뉴스레터가 본문에 표시됩니다.

## 요약본 형식 예시

```
## AI / 개발
### GPT-5 출시 관련 논의
- OpenAI가 새 모델을 공개했으며, 코딩 성능이 대폭 향상됐다는 평가
- 국내 개발자들 사이에서 실무 적용 사례 공유 활발

## 스타트업 / 비즈니스
### 2월 투자 시장 동향
- 시드 단계 투자 감소 추세, 시리즈B 이상 집중 현상
- ...
```

## 자주 묻는 질문

**Q. 여러 채팅방 파일을 업로드할 수 있나요?**
네. 채팅방마다 파일을 따로 업로드하면 됩니다. 요약 생성 시 모든 채팅방 내용이 함께 분석됩니다.

**Q. 요약이 생성되기까지 얼마나 걸리나요?**
채팅량에 따라 다르지만 보통 1~2분 정도 소요됩니다.

**Q. 이미 요약된 날짜를 다시 생성할 수 있나요?**
네, 같은 날짜를 다시 선택해서 생성하면 기존 요약이 교체됩니다.

**Q. 모바일에서도 사용할 수 있나요?**
네. 모바일 화면에서는 상단 햄버거 메뉴(≡)를 눌러 사이드바를 열 수 있습니다.
